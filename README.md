## ✨ 项目介绍

Snet.Iot.Daq 是依托 **Shunnet.top 工业通信库** 开发的插件化采集工具，专为工业设备数据采集场景设计。

支持：

- 多设备并发采集
- 字节级协议解析
- 插件扩展架构
- 点位映射与存储
- 高性能稳定运行

适用于：

- 工业自动化
- PLC 数据采集
- 设备监控系统
- IoT 边缘采集网关
- 自定义协议设备

运行环境：

- 工业现场
- 边缘计算设备
- Windows 服务器

## 📦 安装与使用

### 1️⃣克隆仓库

``` bash
git clone https://github.com/shunnet/Daq.git
cd Daq
```

### 2️⃣ 编译项目

使用 **Visual Studio 2022** 或更高版本打开：

`Snet.Iot.Daq.sln`

选择 Debug 或 Release 构建。

### 3️⃣ 运行程序

构建完成后，在输出目录中找到 `Snet.Iot.Daq.exe`，双击运行即可启动。
